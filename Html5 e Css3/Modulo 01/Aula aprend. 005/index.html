Principais formatações

Negrito / destaque

Podemos usar a tag <strong> para colocar em destaque

Não utlizar mais a tag <b>

Itálico / Enfâse

Podemos usar a tag <em> para deixar em enfâse

Não utilizar mais a tag <i>

Texto marcado

Usamos a tag <mark> para marcar um texto.

Texto grande e pequeno

Para colocar o texto em grande utilizamos a tag <big> (tag já obsoleta) e para colocar o texto em pequeno utilizamos a tag <small>.

Texto deletado

Para marcar um texto como excluido utilizamos a tag <del>

Texto inserido

Para marcar um texto como inserido utilizamos a tag <ins>

Texto sobrescrito

<sup>

Texto subescrito

<sub>
